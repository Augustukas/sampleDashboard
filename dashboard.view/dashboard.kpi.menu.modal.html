Kpi view menu

Create template
  To create new Kpi template click "Create new" button.
  Enter template name then select from "Kpi's to hide" list kpi's
  and click left arrow. Selected kpi's now appears in left
  list named "Kpi's to show".Click "Save" or "Cancel" buttons.

Delete template
  Select template from "Personal templates" list, click "Delete".

Edit template
  Select template from "Personal templates" list, click "Edit".
